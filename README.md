## Overview

This project proposes a system to:
- Detect vehicle accidents using infrared sensors
- Monitor driver drowsiness using eye-blink sensors
- Send emergency alerts using GSM and GPS modules
- Implement the system using LabVIEW and NI myRIO

The system improves response time and reduces fatalities by automatically notifying emergency services.

---

## System Components

- Infrared Sensor → Accident detection  
- Eye Blink Sensor → Drowsiness detection  
- GSM Module → SMS alert  
- GPS Module → Location tracking  
- NI myRIO → Controller  
- LabVIEW → Implementation platform  

---

## Workflow

1. Detect accident using IR sensor  
2. Monitor driver alertness using eye blink sensor  
3. If accident occurs:
   - Capture GPS location  
   - Send SMS via GSM  
4. Trigger alarm if driver is drowsy  

---

## Results

- Real-time accident detection  
- Immediate SMS alert to emergency contacts  
- Effective drowsiness detection system
